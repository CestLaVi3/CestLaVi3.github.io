Linux环境下安装Eclipse开发环境

发布于 2015-07-25 10:03 类别: tech

JDK

先卸载 查 rpm -qa |grep gcj

卸载 yum -y remove java-1.4.2-gcj-compat-src-1.4.2.0-40jpp.115

查看java –version

退出 用root登录

进入Java文件夹下解压文件 tar –zxvf jdk-7u9-linux-i586.tar.gz 解压完毕！

vi ~/.bachrc 在文件中加入这三行

export JAVA_HOME=/usr/java/jdk1.7.0_09

export PATH=$JAVA_HOME/bin:$PATH

export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar

java –version

安装成功！

Tomcat

把tomcat的压缩包放在/usr/local下，解压。 tar -zxvf apache-tomcat-8.0.23.tar.gz

键入命令: /usr/local/apache-tomcat-8.0.23/bin/startup.sh

MySQL

把MySQL的安装包放在/usr下

进入MySQL，安装服务器 rpm -ivh MySQL-server-5.5.19-1.linux2.6.x86.rpm

安装客户端 rpm -ivh MySQL-client-5.5.19-1.linux2.6.x86.rpm

查看服务是否开启 netstat –nat

3306证明服务已开启。 进入mysql

退出 Ctrl+C

Eclipse

在/usr下新建eclipse文件夹并把压缩包放进来。 进入文件夹

开始解压 tar -zxvf eclipse-jee-luna-SR2-linux-gtk.tar.gz 解压完成后进入eclipse打开 ./eclipse
